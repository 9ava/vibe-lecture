HTML/CSS 기초 실습

랜딩 페이지의 기본 구조를 만들어봅니다

HTML CSS

실습 목표

  • HTML 문서의 기본 구조 이해하기
  • 시맨틱 태그 사용법 익히기 (nav, section, footer)
  • CSS 선택자와 속성 사용하기
  • 박스 모델 이해하기 (margin, padding, border)

실습 단계

Step 1: HTML 기본 구조

DOCTYPE, html, head, body 태그로 문서 구조를 만듭니다.

Step 2: 네비게이션 만들기

nav 태그 안에 로고와 메뉴 링크를 추가합니다.

Step 3: 히어로 섹션

큰 제목과 설명, CTA 버튼이 있는 섹션을 만듭니다.

Step 4: CSS 스타일 적용

색상, 폰트, 여백 등 기본 스타일을 적용합니다.

시작 코드

아래 코드를 복사해서 시작하세요:

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Page</title> <style> body { font-family: sans-serif; margin: 0; padding: 0; } nav { background: #6366f1; color: white; padding: 1rem; } .hero { text-align: center; padding: 100px 20px; background: #f0f0f0; } .hero h1 { font-size: 3rem; margin-bottom: 1rem; } .btn { display: inline-block; padding: 1rem 2rem; background: #6366f1; color: white; text-decoration: none; border-radius: 8px; } </style> </head> <body> <nav> <span>My Logo</span> </nav> <section class="hero"> <h1>Welcome!</h1> <p>나만의 웹페이지를 만들어보세요</p> <a href="#" class="btn">시작하기</a> </section> </body> </html>

도전 과제

  • 네비게이션에 메뉴 링크 3개 추가하기
  • 히어로 섹션 배경에 그라데이션 적용하기
  • 버튼에 hover 효과 추가하기
  • 푸터 섹션 추가하기
