Backend가 필요한 사이트 만들기

서버와 데이터베이스가 필요한 웹 서비스를 만들어봅니다

Backend Database

강의 개요

Frontend만으로는 구현할 수 없는 기능들(로그인, 데이터 저장, 사용자 간 상호작용 등)을 Backend를 통해 구현하는 방법을 배웁니다. 서버, 데이터베이스, API의 개념을 이해하고 실제로 동작하는 풀스택 웹 서비스를 만들어봅니다.

학습 내용

01

Frontend vs Backend

Frontend(클라이언트)와 Backend(서버)의 역할 차이를 이해하고, 언제 Backend가 필요한지 알아봅니다.

02

서버와 데이터베이스

서버가 하는 일과 데이터베이스의 역할을 배웁니다. 데이터를 저장하고 불러오는 기본 원리를 이해합니다.

03

Supabase 소개

코드 없이 Backend를 구축할 수 있는 Supabase를 소개합니다. 데이터베이스, 인증, 스토리지를 쉽게 사용하는 방법을 배웁니다.

04

실습: 방명록 만들기

Supabase를 활용해 실제로 데이터를 저장하고 불러오는 간단한 방명록 웹사이트를 만들어봅니다.

Backend가 필요한 경우

  • 데이터 저장 - 사용자가 작성한 글, 댓글, 프로필 정보 등
  • 로그인/회원가입 - 사용자 인증과 계정 관리
  • 사용자 간 상호작용 - 채팅, 좋아요, 팔로우 등
  • 파일 업로드 - 이미지, 동영상 등의 저장
  • 보안이 필요한 작업 - 결제, 민감한 데이터 처리

Supabase vs Railway 비교

Backend를 쉽게 구축할 수 있는 두 가지 서비스를 비교해봅니다.

Supabase

BaaS (Backend as a Service) - 백엔드 기능을 서비스로 제공

  • 특징: 코드 작성 없이 백엔드 구축 가능
  • 제공 기능: PostgreSQL 데이터베이스, 인증(로그인), 파일 스토리지, 실시간 구독
  • 장점: 빠른 시작, 직관적인 대시보드, 무료 티어 제공
  • 단점: 복잡한 비즈니스 로직 구현에 한계
  • 적합한 경우: 간단한 CRUD 앱, MVP, 프로토타입, 개인 프로젝트

Railway

PaaS (Platform as a Service) - 서버 배포 플랫폼

  • 특징: 직접 작성한 백엔드 코드를 쉽게 배포
  • 지원: Node.js, Python, Go, Rust 등 다양한 언어/프레임워크
  • 장점: 자유로운 커스터마이징, 복잡한 로직 구현 가능
  • 단점: 백엔드 코드 작성 필요, 상대적으로 학습 필요
  • 적합한 경우: 복잡한 비즈니스 로직, 커스텀 API, 확장성 필요한 서비스

한눈에 비교

구분 Supabase Railway
유형 BaaS (Backend as a Service) PaaS (Platform as a Service)
코딩 필요 거의 없음 (SDK 호출만) 백엔드 코드 직접 작성
데이터베이스 PostgreSQL (내장) PostgreSQL, MySQL, Redis 등 선택
인증 기능 내장 (소셜 로그인 포함) 직접 구현 필요
자유도 낮음 (정해진 기능 사용) 높음 (원하는 대로 구현)
난이도 쉬움 중간
무료 티어 2개 프로젝트, 500MB DB 월 $5 크레딧

어떤 걸 선택할까?

  • Supabase 추천: 빠르게 시작하고 싶거나, 백엔드 코딩 경험이 없는 경우
  • Railway 추천: 복잡한 로직이 필요하거나, 백엔드 개발을 배우고 싶은 경우
  • 함께 사용: Supabase(DB/인증) + Railway(복잡한 API)로 조합도 가능!

예제 코드

Supabase로 데이터를 저장하고 불러오는 예시입니다:

데이터 불러오기 (Read)

// 방명록 데이터 불러오기 const { data, error } = await supabase .from('guestbook') .select('*') .order('created_at', { ascending: false }); console.log(data); // 저장된 방명록 목록

데이터 저장하기 (Create)

// 새 방명록 작성하기 const { data, error } = await supabase .from('guestbook') .insert([ { name: '홍길동', message: '안녕하세요!' } ]); if (!error) { console.log('저장 성공!'); }
