웹 개발 기초 실습

HTML과 CSS로 랜딩 페이지를 직접 만들어봅니다

실습 목표

이번 실습에서는 다음을 직접 만들어봅니다:

  • 히어로 섹션이 있는 랜딩 페이지
  • Flexbox를 활용한 네비게이션
  • CSS Grid로 구성한 카드 레이아웃
  • 반응형 디자인 적용

실습 단계

Step 1: 프로젝트 설정

새 폴더를 만들고 index.html 파일을 생성합니다.

Step 2: HTML 구조 작성

기본 HTML 문서 구조와 시맨틱 태그를 작성합니다.

Step 3: CSS 스타일링

네비게이션, 히어로 섹션, 카드 레이아웃에 스타일을 적용합니다.

Step 4: 반응형 적용

미디어 쿼리를 사용해 모바일 화면에 대응합니다.

기본 HTML 템플릿

아래 코드를 복사해서 시작하세요:

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My Landing Page</title> <style> /* 여기에 CSS 작성 */ </style> </head> <body> <nav> <!-- 네비게이션 --> </nav> <section class="hero"> <!-- 히어로 섹션 --> </section> <section class="features"> <!-- 기능 소개 카드 --> </section> </body> </html>
