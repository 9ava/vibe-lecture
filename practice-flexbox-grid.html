Flexbox & Grid 실습

반응형 레이아웃을 만들어봅니다

CSS Flexbox Grid

실습 목표

  • Flexbox로 1차원 레이아웃 만들기
  • CSS Grid로 카드 레이아웃 구성하기
  • 미디어 쿼리로 반응형 적용하기

Flexbox 기초

Flexbox는 1차원(가로 또는 세로) 레이아웃에 적합합니다.

Flexbox 예시

1
2
3
.container { display: flex; justify-content: space-between; /* 가로 정렬 */ align-items: center; /* 세로 정렬 */ gap: 1rem; /* 간격 */ }

주요 속성

  • justify-content: 메인 축 정렬 (flex-start, center, space-between)
  • align-items: 교차 축 정렬 (flex-start, center, stretch)
  • flex-direction: 방향 (row, column)
  • gap: 아이템 간격

CSS Grid 기초

Grid는 2차원(행과 열) 레이아웃에 적합합니다.

Grid 예시

1
2
3
4
5
6
.container { display: grid; grid-template-columns: repeat(3, 1fr); /* 3열 균등 분배 */ gap: 1rem; } /* 반응형: 모바일에서 1열 */ @media (max-width: 768px) { .container { grid-template-columns: 1fr; } }

주요 속성

  • grid-template-columns: 열 정의
  • grid-template-rows: 행 정의
  • gap: 행/열 간격
  • repeat(): 반복 패턴
  • 1fr: 남은 공간 균등 분배

도전 과제

  • Flexbox로 네비게이션 만들기 (로고 왼쪽, 메뉴 오른쪽)
  • Grid로 3열 카드 레이아웃 만들기
  • 768px 이하에서 1열로 변경되는 반응형 적용
  • auto-fit과 minmax()로 유동적 그리드 만들기
