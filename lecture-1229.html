웹 개발 기초

HTML과 CSS로 나만의 웹페이지를 만들어봅니다

HTML CSS

강의 개요

웹 개발의 기초가 되는 HTML과 CSS를 배웁니다. 실습을 통해 간단한 랜딩 페이지를 직접 만들어보고, 반응형 레이아웃의 기본 개념을 익힙니다.

학습 내용

01

HTML 기초

HTML 문서 구조, 시맨틱 태그, 폼 요소 등 웹페이지의 뼈대를 만드는 방법을 배웁니다.

02

CSS 스타일링

선택자, 박스 모델, 색상, 폰트 등 웹페이지를 꾸미는 기본 CSS 속성들을 학습합니다.

03

Flexbox 레이아웃

유연한 1차원 레이아웃을 만드는 Flexbox의 핵심 개념과 속성을 배웁니다.

04

CSS Grid

복잡한 2차원 레이아웃을 쉽게 만드는 CSS Grid 시스템을 학습합니다.

예제 코드

기본적인 HTML 구조 예시입니다:

<!DOCTYPE html>
<html lang="ko">
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>
